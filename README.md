# 🎓 English Master - 大人の英語学習ゲーム

インタラクティブで楽しい大人向けの英語学習ゲームです。3つの異なるゲームモードで英語力を向上させましょう！

## ✨ 特徴

- **3つのゲームモード**
  - 📚 語彙クイズ - ビジネス英語の重要単語を学習
  - ✍️ 文章構築 - 単語を並べて正しい英文を作成
  - 🌐 翻訳チャレンジ - 日本語を英語に翻訳

- **学習機能**
  - リアルタイムのスコア追跡
  - 連続正解数の記録
  - プログレスバーで進捗確認
  - 即座のフィードバックとヒント

- **レスポンシブデザイン**
  - モバイル・タブレット・デスクトップ対応
  - 美しいグラデーションUI
  - スムーズなアニメーション

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 好きなゲームモードを選択
3. 問題に答えてスコアを伸ばす！

## 💻 技術スタック

- HTML5
- CSS3（アニメーション・グラデーション）
- Vanilla JavaScript（フレームワーク不要）

## 📦 インストール

```bash
# リポジトリをクローン
git clone https://github.com/omix222/english-learning-game.git

# ディレクトリに移動
cd english-learning-game

# ブラウザで開く
open index.html  # macOS
start index.html # Windows
```

## 🎯 学習内容

### 語彙クイズ
- Accommodate（適応させる）
- Ambiguous（曖昧な）
- Pragmatic（実用的な）
- Resilient（回復力のある）
- その他多数のビジネス英語

### 文章構築
- 現在完了進行形
- 未来完了形
- 関係代名詞を使った文章

### 翻訳チャレンジ
- 日常会話からビジネス表現まで
- ヒント機能付き
